2. **Clone your fork** locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/research-seminar-project.git
   ```
3. **Create a branch** for your feature or fix:
   ```bash
   git checkout -b feature/amazing-feature
   ```
4. **Implement your changes** following the project's coding standards.
5. **Ensure your code passes linting**:
   ```bash
   cd backend && npm run lint
   cd ../frontend && npm run lint
   ```
6. **Commit your changes**:
   ```bash
   git commit -m "Add: Descriptive message about the change"
   ```
7. **Push to your fork** and **open a Pull Request**.
